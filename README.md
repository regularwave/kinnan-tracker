# Kinnan Tracker

Mana pool, life, commander tax, and commander damage tracking!

## Description

A slightly more robust and Kinnan-themed way of tracking mana pools, life, commander tax, and commander damage. 

## Getting Started

### Installing

* **Browser:** open the [Kinnan Tracker](https://regularwave.github.io/kinnan-tracker/) page. That's it!
* **App (PWA install):**
  * **Android**
    1. Open the [Kinnan Tracker](https://regularwave.github.io/kinnan-tracker/) page in **Chrome**.
    2. Tap the **three-dot menu** icon in the top-right corner.
    3. Select **Install app** or **Add to Home Screen**.
    4. Follow the on-screen prompt to install.
  * **iOS**
    1. Open the [Kinnan Tracker](https://regularwave.github.io/kinnan-tracker/) page in **Safari**.
    2. Tap the **Share** button (square with an arrow pointing up) at the bottom of the screen.
    3. Scroll down the list and tap **Add to Home Screen**.
    4. Tap **Add** in the top-right corner.

## Help

Check the Help link at the bottom of the [Kinnan Tracker](https://regularwave.github.io/kinnan-tracker/) page for usage instructions.
